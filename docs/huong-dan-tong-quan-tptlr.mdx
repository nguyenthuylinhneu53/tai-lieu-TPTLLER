        <li><strong>Đường dẫn thêm quỹ tiền thủ công:</strong>
          <ul>
            <li><em>Tại phân hệ Giao dịch ngân quỹ:</em> Chọn <code>Thêm quỹ tiền</code> → <code>Lựa chọn quỹ tiền</code></li>
            <li><em>Tại phân hệ Giao dịch khách hàng:</em> Chọn <code>Thêm quỹ tiền</code> → <code>Lựa chọn quỹ tiền</code></li>
          </ul>
        </li>
      </ul>

      <h3>3.4 Menu Tiện ích / Utilities</h3>
      <ul>
        <li><strong>Khóa ứng dụng (Lock Application):</strong> Khóa nhanh màn hình làm việc của TP Teller tạm thời để bảo mật dữ liệu khi GDV rời vị trí.</li>
        <li><strong>Máy tính (Calculator):</strong> Tích hợp 03 cấu hình nhập liệu chuyên dụng:
          <ul>
            <li><em>Tiêu chuẩn (Standard):</em> Tính toán số học cơ bản.</li>
            <li><em>Nhập theo số tiền (Denomination by Amount):</em> Nhập tổng số tiền tương ứng với từng mệnh giá khai báo để tự động tính tổng bảng kê tiền mặt.</li>
            <li><em>Nhập theo số tờ (Denomination by Count):</em> Kiểm đếm bằng cách nhập tổng số tờ theo từng mệnh giá để hệ thống nhân quy đổi ra tổng giá trị trị bảng kê.</li>
          </ul>
        </li>
        <li><strong>Máy tính quy đổi ngoại tệ (Exchange Cal):</strong> Hỗ trợ quy đổi dòng tiền giữa các loại tiền tệ dựa trên tỷ giá mua/bán realtime từ hệ thống.
          <br />⚠️ <em>Lưu ý nghiệp vụ:</em> Nếu NSD chỉnh sửa tỷ giá vượt quá biên độ cho phép được cấu hình từ TSC, chương trình sẽ báo lỗi block giao dịch ngay lập tức.
        </li>
        <li><strong>Trạng thái host (Host Status):</strong> Kiểm tra trạng thái kết nối Online/Offline của máy trạm kết nối với Máy chủ Trung ương. Khi trạng thái Host bị mất kết nối (Offline), dòng chữ cảnh báo <code>offline</code> màu đỏ sẽ hiển thị cố định trên thanh Taskbar phía dưới màn hình.</li>
      </ul>

      <h3>4.1 Sử dụng tổ hợp phím tắt "CTRL / ALT + Chữ cái"</h3>
      <ul>
        <li><kbd>CTRL</kbd> + <kbd>Chữ cái</kbd>: Áp dụng để kích hoạt mở nhanh các phân hệ chính từ trang chủ:
          <ul>
            <li><kbd>Ctrl + B</kbd>: Mở nhanh mục Quản trị Chi nhánh (Branch Maint).</li>
            <li><kbd>Ctrl + K</kbd>: Mở nhanh phân hệ Vấn tin và duy trì séc.</li>
            <li><kbd>Ctrl + U</kbd>: Mở nhanh phân hệ Giao dịch Ngân quỹ.</li>
            <li><kbd>Ctrl + T</kbd>: Mở nhanh phân hệ Giao dịch với Khách hàng.</li>
          </ul>
        </li>
        <li><kbd>ALT</kbd> + <kbd>Chữ cái</kbd>: Áp dụng để thực thi lệnh nhanh cho các nút chức năng (Button) nằm bên trong nội dung màn hình giao dịch cụ thể.</li>
      </ul>

      <br />
      <h3>4.2 Quy tắc di chuyển vùng nhập liệu bằng bàn phím</h3>
      <ul>
        <li>Sử dụng phím <kbd>Tab</kbd> hoặc lệnh Shift + Tab để di chuyển tuần tự giữa các trường thông tin dữ liệu (Input fields) và các nút lệnh trên form.</li>
        <li><strong>Thao tác với danh sách lựa chọn (Dropdown List):</strong> Khi con trỏ chuột di chuyển đến trường Dropdown, nhấn phím <kbd>F4</kbd> trên bàn phím để hệ thống tự động bung danh sách (Sổ xuống). Sau đó sử dụng cụm phím <kbd>↑ Mũi tên lên</kbd> / <kbd>↓ Mũi tên xuống</kbd> để di chuyển chọn giá trị cần hạch toán.</li>
        <li><strong>Đối với bàn phím có cụm phím số độc lập (Numpad):</strong> NSD có thể sử dụng phím dấu trừ <kbd>-</kbd> hoặc phím dấu cộng <kbd>+</kbd> tại cụm Numpad để nhảy nhanh giữa các ô nhập liệu thay thế cho phím Tab, tối ưu hóa tốc độ nhập liệu của Giao dịch viên.</li>
      </ul>
